HTML5

HTML (Linguagem de Marcação de Hipertexto) é uma linguagem de marcação usada para definir a estrutura do conteúdo de uma página webEla consiste em uma série de elementos que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira. Aqui estão alguns dos principais aprendizados em HTML:

Estrutura básica: A estrutura básica de um documento HTML inclui as tags html,head e body. O conteúdo da página é colocado dentro da tag body.

Tags: As tags são usadas para delimitar diferentes partes do conteúdo. Algumas tags comuns incluem p para parágrafos, h1 a h6 para títulos e subtítulos, ul e ol para listas, a para links e img para imagens.

Atributos: Os atributos são usados para fornecer informações adicionais sobre as tags. Por exemplo, o atributo href é usado para especificar o URL de um link, enquanto o atributo src é usado para especificar o URL de uma imagem.

Formulários: Os formulários são usados para coletar informações do usuário.Eles podem incluir campos de texto, botões de opção, caixas de seleção e botões de envio.

Tabelas: As tabelas são usadas para exibir informações em um formato tabular.
